Output
There are 3 different ways to Output values in JavaScript.

Using document.write("text") this method replaces all the content in webpage with the "text" we provided. Try the following code in a HTML file (We recommend you use Spck editor for android).

Example

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<head>
<script>
document.write("Hello World")
</script>
</head>
<body>
</body>
</html>
We will not use this method that much in our practices.

Using alert("text") this method opens a popup window on client side and shows our output.Unlike the previous method, this doesnt erase our webpage content

Example

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<head>
<script>
alert("Hello World");
</script>
</head>
<body>
</body>
</html>

Using console.log(text). This is the method most developers use for debugging or testing purpose. This method writes the logs or texts in our browser's console.

Example

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<head>
<script>
console.log("Hello World");
</script>
</head>
<body>
</body>
</html>
Copy the above code and paste it in a HTML file. Use SPCK editor for android. Run the code and then the webpage will be displayed. In top-right corner click on the three dots and then enable "console" in the settings. You will see a console window opened and in the console you will see our output.

These are the methods we use to output texts in JavaScript. In our tutorial we will use alert() method the most.

Input
There are few methods to ask user for their input but the most common and easy method is the prompt method.
prompt(text) method display a text message and ask for user's input. and we can store the input in a variable.

Example

let myInput = prompt("Enter your input");
alert("You entered " + myInput);
