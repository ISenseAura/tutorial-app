JavaScript can be used inside of a HTML file or outside the HTML file. To use JavaScript inside your HTML you'll have to add a script tag. जावास्क्रिप्टचा उपयोग HTML फाइलमध्ये किंवा HTML फाइलच्या बाहेर केला जाऊ शकतो. आपल्या HTML मध्ये JavaScript वापरण्यासाठी आपल्याला script टॅग जोडावे लागेल. जावास्क्रिप्ट का उपयोग HTML फ़ाइल के अंदर या HTML फ़ाइल के बाहर किया जा सकता है। अपने HTML के अंदर जावास्क्रिप्ट का उपयोग करने के लिए आपको एक script टैग जोड़ना होगा।

Example

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<head>
<script>
alert("Hello World");
</script>
</head>
<body>
</body>
</html>
Now create a file using any text editor and paste the above code inside it and save the file as "index.html" and then open the file. You will get a popup window saying "Hello World" as output.

You can also put your JavaScript code in a seperate file and run it into your HTML file. for that we use the attribute src. For example ,If we wanna use a JavaScript file named "file.js" inside our HTML file and both files are in same folder then we use <script src="file.js"> </script>. inside src attribute there goes our file path or location.

The script tag can be used inside the head or body tags. आता कोणताही टेक्स्ट एडिटर वापरून फाईल बनवा आणि त्यामध्ये वरील कोड पेस्ट करा आणि फाईल "index.html" म्हणून सेव्ह करा आणि मग फाईल उघडा. आउटपुट म्हणून आपल्याला "Hello World" म्हणणारी पॉपअप विंडो मिळेल.

आपण आपला जावास्क्रिप्ट कोड वेगळ्या फाईलमध्ये ठेवू शकता आणि आपल्या HTML फाइलमध्ये चालवू शकता. त्यासाठी आम्ही src attribute वापरतो. उदाहरणार्थ, आम्ही आमच्या HTML फाइलमध्ये "file.js" नावाची जावास्क्रिप्ट फाइल वापरू इच्छित असाल आणि दोन्ही फायली एकाच फोल्डरमध्ये असतील तर आम्ही <script src="file.js"> </script> $$$$$. src attribute मध्ये आमचा फाईल मार्ग किंवा location आहे.
